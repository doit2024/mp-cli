# mp-cli

- 生成多页应用脚手架的工具
- 模板地址： https://github.com/niuDazhuang/webpack4_demo

---

## install

```
$ npm i -g mp-cli
```

## start

```
$ mp project-name
```

## run

```bash
$ npm start
# or
$ npm run dev

# visit
# => http://localhost:8080/record.html
# => http://localhost:8080/statistic.html
# => http://localhost:8080/xinlv.html
```

## build

```
$ npm run build
```
